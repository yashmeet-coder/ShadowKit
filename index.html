ShadowKit

An embeddable widget scaffold built with React, shadcn/ui and the Shadow DOM. Drop it onto any page with a single <script> tag — its styles never leak out, and the host page's styles never leak in.

Style isolation

Everything on this page is styled by the host page. The Feedback button in the bottom-right corner is not — it lives inside a shadow root, so its styling is completely independent of this page's CSS. For comparison, here is an ordinary host-page button:
